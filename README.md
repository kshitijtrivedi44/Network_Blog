# Network_Blog
Technical blog for Network VA/PT 

As organizations increasingly rely on interconnected systems for communication, data storage, and business operations, securing network infrastructure has become a critical aspect of cybersecurity. Every device connected to a network—whether a workstation, server, router, or IoT device—can become a potential entry point for attackers if not properly secured. Identifying these weaknesses before they are exploited is therefore an essential part of maintaining a secure environment.
Vulnerability Assessment (VA) and Penetration Testing (PT) are two widely adopted security assessment techniques used to evaluate the security posture of a network. Although they are often mentioned together, they serve different purposes. A Vulnerability Assessment focuses on identifying and prioritizing security weaknesses, whereas a Penetration Test attempts to safely validate whether those weaknesses can actually be exploited under controlled conditions. Together, they help organizations understand their attack surface, evaluate risks, and strengthen their overall security posture.
This document adopts a practical approach to Network VA/PT by performing a security assessment in a controlled virtual lab environment. Rather than focusing only on theoretical concepts, the report demonstrates how commonly used security tools can be applied to discover hosts, identify open ports, enumerate running services, analyze network traffic, and identify potential security risks. Each activity is supported with observations, analysis, and security implications to reflect the workflow followed during a real network assessment.
The objective of this report is not only to understand the tools involved in a network security assessment but also to interpret their results from a security perspective. By combining hands-on experimentation with research on identified findings, the report aims to demonstrate how network reconnaissance and vulnerability assessment contribute to identifying and mitigating potential security threats.
